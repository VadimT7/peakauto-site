Conectare (o singură dată)

Panoul salvează totul în VadimT7/peakauto-site pe GitHub — baza de date a site-ului. După fiecare „Publică pe site", site-ul se actualizează automat în ~1 minut.

  1. GitHub → Settings → Developer settings → Fine-grained tokens → Generate new token
  2. Repository access: VadimT7/peakauto-site
  3. Permissions → Contents: Read and write

Tokenul rămâne doar în acest browser. Nu se trimite nicăieri în afară de api.github.com.

Apasă Editează pentru orice mașină — preț, date, descriere, dotări, poze. ★ = intră în caruselul „În prim-plan". Statusul „Vândut" lasă mașina pe site cu banda SOLD, „Ascuns" o scoate complet. Nimic nu ajunge pe site până nu apeși Publică pe site.
