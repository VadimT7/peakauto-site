Conectare la GitHub

Panoul salvează modificările direct în repository-ul VadimT7/peakauto-site. Site-ul se republică automat în ~1 minut după fiecare publicare.

  1. GitHub → Settings → Developer settings → Fine-grained tokens → Generate new token
  2. Repository access: VadimT7/peakauto-site
  3. Permissions: Contents → Read and write, Actions → Read and write

Tokenul rămâne doar în acest browser (localStorage). Nu se trimite nicăieri în afară de api.github.com.
